# 等级难度设计理念

*创建：2026-04-28 | 版本：v1.0*

---

## 核心原则：轨道决定基准，等级决定进度

每个轨道面向的用户群体不同，其"入门"起点天然不同。  
L1 代表的是**该轨道内的入门**，而非全局英语水平的入门。

```
旅行家 L1 ≠ 应试派 L1（但都是各自轨道内最简单的起点）
```

---

## 三个轨道的难度基准

| 轨道 | 目标用户 | L1 起点 | 词汇来源 |
|------|----------|---------|---------|
| 🌍 旅行家 | 有出行需求，英语基础不限 | 日常高频词（COCA Top 2000） | 旅行场景词 + 生活常用词 |
| 🎬 追剧党 | 喜欢美剧，有感性基础 | 日常口语高频词（COCA Top 2000） | 美剧高频词 + 情感表达 |
| 📚 应试派 | 备考四六级的学生 | 四级核心高频词（COCA 2000–8000） | 四级大纲词 + 考试高频词 |

**设计逻辑**：选择"应试派"的用户已明确表达自己在备考，默认具备基础词汇量（hello、family 这类词无需再学）。从四级核心词开始才是对用户时间的尊重。

## 六个等级的难度维度

难度由以下五个维度共同决定，等级越高，各维度要求越高：

### 1. 词频（Word Frequency）
最客观的难度指标，使用 COCA（美国当代英语语料库）词频排名。

| 等级 | COCA 词频区间 | 说明 |
|------|-------------|------|
| L1 | 旅/剧：Top 2000 / 试：2000–6000 | 超高频，随处可见 |
| L2 | 旅/剧：2000–4000 / 试：4000–8000 | 高频，日常常用 |
| L3 | 旅/剧：4000–7000 / 试：6000–12000 | 中频，需要积累 |
| L4 | 旅/剧：6000–10000 / 试：10000–18000 | 低频，需要主动学习 |
| L5 | 10000–18000 | 较低频，考试/专业场景 |
| L6 | 18000+ | 低频，进阶/地道表达 |

### 2. 词义复杂度（Semantic Complexity）

| 等级 | 特征 | 示例 |
|------|------|------|
| L1 | 单义词、具体词、中英对应直观 | suitcase = 行李箱 |
| L2 | 开始出现常见多义 | book = 书 / 预订 |
| L3 | 多义词，需结合语境判断 | hold = 抓住 / 举行 / 容纳 |
| L4 | 抽象词，词义偏学术 | facilitate = 促进 |
| L5 | 高度抽象，需语境才能准确理解 | inherent = 固有的 |
| L6 | 地道用法、引申义、文化负载 | bear = 忍受 / 生育 / 承担 |

### 3. 音节与拼写（Phonetic Complexity）

| 等级 | 平均音节数 | 典型示例 |
|------|----------|---------|
| L1 | 1–2 音节 | go, happy, ticket |
| L2 | 2–3 音节 | journey, arrive, payment |
| L3 | 2–3 音节 | consider, develop, effective |
| L4 | 3–4 音节 | facilitate, evaluate, demonstrate |
| L5 | 3–4 音节 | circumstance, sophisticated, controversy |
| L6 | 4+ 音节 | idiosyncratic, comprehensive, unprecedented |

### 4. 记忆策略（Memory Strategy Required）

| 等级 | 记忆方式 |
|------|---------|
| L1 | 看图/场景即懂，中英对应简单直接 |
| L2 | 需要简单语境辅助记忆 |
| L3 | 依赖例句和使用场景 |
| L4 | 需要词根分解（如 ac-quire, re-quire） |
| L5 | 需辨析近义词（如 affect vs effect） |
| L6 | 需理解搭配、语域、文化语境 |

### 5. 考试标准对应（Exam Standard Mapping）

仅适用于应试派轨道，作为辅助参考：

| 等级 | 对应考试标准 | ECDICT tag |
|------|------------|-----------|
| L1 | 四级基础（高频考纲词） | cet4（高频部分） |
| L2 | 四级全覆盖 | cet4 |
| L3 | 四级难词 + 六级入门 | cet4（低频）+ cet6（高频） |
| L4 | 六级核心 | cet6 |
| L5 | 六级难词 + 雅思 | cet6（低频）+ ielts |
| L6 | 托福 / GRE | toefl / gre |

## 各轨道的六级路线图

### 🌍 旅行家
```
L1 行李机场  →  L2 住宿交通  →  L3 饮食购物
L4 文化习俗  →  L5 商务出行  →  L6 地道口语
```

### 🎬 追剧党
```
L1 问候情绪  →  L2 家庭恋爱  →  L3 职场社交
L4 剧情描述  →  L5 地道俚语  →  L6 文化典故
```

### 📚 应试派
```
L1 四级高频  →  L2 四级全覆  →  L3 四六过渡
L4 六级核心  →  L5 六级难词  →  L6 托福/GRE
```

## 词表设计红线

**L1 禁止出现的词类型：**
- 低频学术词（accumulate, idiosyncrasy）
- 专业术语（paradigm, heuristic）
- 拼写极不规则的词（pneumonia, phlegm）

**L1 应优先包含的词类型：**
- 高频动词（develop, improve, consider）
- 高频名词（situation, purpose, effect）
- 日常场景词（ticket, hotel, meeting）

## 数据生成策略

脚本 `scripts/generate-level-data.mjs` 从以下数据源获取词汇信息：

| 数据源 | 提供内容 | 对应难度筛选字段 |
|--------|---------|----------------|
| ECDICT（35万词离线库） | 音标、中文释义、词性 | `tag`（考试分级）、`collins`（星级）、`frq`（COCA频率） |
| Free Dictionary API | 英文例句 | — |

后续生成更高等级词表时，脚本应优先筛选 ECDICT 中 `tag` 字段匹配对应等级的词汇，再按 `frq` 排序取高频部分。

## 解锁机制与进度系统

完成当前等级 **75%** 的关卡后解锁下一等级（可在 `src/data/levels.ts` 中通过 `UNLOCK_THRESHOLD` 调整）。

| 等级 | 关卡数 | 解锁所需完成关卡 |
|------|--------|----------------|
| L1 | 40 | 注册即解锁 |
| L2 | 60 | 完成 L1 共 30 关 |
| L3 | 80 | 完成 L2 共 45 关 |
| L4 | 100 | 完成 L3 共 60 关 |
| L5 | 100 | 完成 L4 共 75 关 |
| L6 | 120 | 完成 L5 共 75 关 |
