# UI/UX 优化计划 - v1.0 （首页、赛道、我的页面）

## 📋 概述

基于用户体验反馈，对首页、赛道、我的三个核心页面进行全面优化。

**目标**：
- 降低首页的认知负荷
- 增强赛道选择的可视性和操作便捷性
- 完善个人页面的信息展示和功能

---

## 🎨 优化方案详情

### 1️⃣ 首页优化（HomePage）

#### 当前问题
- 信息过多：推荐舞台 + 复习提醒 + 4个统计卡片 + 今日推荐词汇 + 打卡日历
- 焦点不清：用户打开不知道先干什么
- 信息竞争：推荐舞台和进入赛道按钮功能重复

#### 优化方案

**布局重构**：
```
┌─────────────────────────────────────┐
│ 顶部：用户欢迎                        │
│ 👋 早上好，yunfan！                  │
│ Lv3 · 总XP 450 · 连续打卡 7 天        │
├─────────────────────────────────────┤
│ 核心操作区（3选1）：                   │
│ ┌──────────────────────────────────┐ │
│ │ 📍 下一个舞台（推荐）              │ │
│ │ 🎬 追剧党 · 情绪表达              │ │
│ │ [立即学习] →                      │ │
│ └──────────────────────────────────┘ │
│                                      │
│ 或者：[🎲 随机挑战]  [📖 选择赛道]   │
├─────────────────────────────────────┤
│ 复习提醒（如果有）：                   │
│ ⏰ 有 5 个词需要复习 [立即复习 →]    │
├─────────────────────────────────────┤
│ 底部：关键数据（3个）                  │
│ ⭐ 本周学词：32    📊 总完成：12关     │
│ 🔥 连续打卡：7 天                     │
└─────────────────────────────────────┘
```

**删除项**：
- ❌ "今日推荐"词汇流（无用，用户不会直接学单词）
- ❌ "主修赛道"卡片（和"选择赛道"重复）
- ❌ "其他赛道"按钮组（可在赛道页面选）
- ❌ 打卡日历（移到"我的"页面）

**保留和增强**：
- ✅ 推荐舞台卡片（增大，突出）
- ✅ 复习提醒（红色高亮）
- ✅ 欢迎语+快速数据条（顶部）
- ✅ 3个操作按钮（推荐舞台 / 随机挑战 / 选择赛道）

---

### 2️⃣ 赛道页面优化（TracksPage）

#### 当前问题
- 太简陋：只显示赛道名称+等级+XP
- 信息缺失：没有赛道描述、没有进度条、没有快捷操作
- 交互不足：只有"进入"一个操作

#### 优化方案

**赛道卡片重新设计**：
```
┌──────────────────────────────────┐
│ 🎬 追剧党                          │
│ 美剧电影里的日常表达                │
├──────────────────────────────────┤
│ 进度：■■■■■□□□□□ 50%  (5/10 关)  │
│ 等级：Lv2  |  经验：150 XP         │
├──────────────────────────────────┤
│ [进入轨道] [继续学习]              │
└──────────────────────────────────┘
```

**具体改动**：
1. 增加赛道描述（原有 track.description）
2. 加进度条：显示完成度百分比 + 完成关卡数
3. 并列两个按钮：
   - "进入轨道"：进入轨道地图
   - "继续学习"：直接进入下一个未完成的舞台（快捷）
4. 卡片内增加视觉层级（用颜色区分赛道）

**页面布局**：
```
├─ 标题：我的赛道
├─ 选中的赛道（2-3个）：完整卡片
│  ├─ 追剧党卡片
│  ├─ 旅行家卡片
│  └─ 应试派卡片
├─ 未选中的赛道：灰显，点击可切换
└─ 底部：[+ 添加新赛道] 按钮
```

### 3️⃣ 我的页面优化（MePage）

#### 当前问题
- 信息缺失：没有显示用户头像、昵称、等级、总XP
- 功能不完整："词包"功能未实现
- 统计不足：没有学习统计数据（本周学词、连续打卡等）
- 缺少个性化：不能编辑头像和昵称

#### 优化方案

**用户信息卡（新增）**：
```
┌──────────────────────────────────┐
│ 🦊 yunfan                        │
│ 总等级：Lv3  |  总XP：450         │
│ 连续打卡：7 天  |  本周学词：32    │
│ [编辑头像] [编辑昵称]            │
└──────────────────────────────────┘
```

**学习统计卡（新增）**：
```
┌──────────────────────────────────┐
│ 📊 本周学习统计                    │
├──────────────────────────────────┤
│ 总词汇：32 个     总关卡：8 关     │
│ 平均准确率：85%   平均星级：★★    │
│ 获得XP：160 点    打卡天数：7 天   │
└──────────────────────────────────┘
```

**菜单项（简化）**：
```
├─ 赛道选择（点击管理选了哪些赛道）
├─ 学习设置
│  ├─ 语音播放速度
│  ├─ 每日目标
│  └─ 提醒设置
├─ 打卡日历（展示本月的打卡情况）
└─ 关于/反馈
```

**移除项**：
- ❌ "词包"菜单项（暂时没实现）

## 📐 设计原则

1. **信息层级清晰**：核心操作突出，辅助信息在下
2. **减少认知负荷**：删除无用信息，集中注意力
3. **一致的卡片设计**：所有卡片使用统一的样式和交互
4. **快捷操作**：常用操作（继续学习、随机挑战）易达

- [ ] 首页整洁，焦点清晰，用户打开即知道干什么

- [ ] 我的页面展示个人成就，有学习动力
- [ ] 所有页面响应快速，动画流畅
- [ ] 移动端适配良好（PWA）
